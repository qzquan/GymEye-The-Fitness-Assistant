
## 项目简介

GymEye 是一款基于 AI 图像识别技术的健身器械助手。用户只需使用手机摄像头扫描健身房中的器械，系统即可自动识别器械类型，并展示对应的名称、训练部位、使用步骤、注意事项以及标准教学视频。

它主要面向健身新手和不熟悉器械的人群，帮助用户快速了解器械用途，避免错误使用，提高训练效率和健身体验。

Slogan：

> GymEye —— 一扫即懂的健身助手

---

## 项目目标

GymEye 希望解决健身新手在第一次进入健身房时经常遇到的问题：

* 不认识器械
* 不知道该怎么调节座椅和配重
* 不知道动作是否标准
* 不敢直接尝试，担心出错或受伤
* 需要花费大量时间在短视频平台搜索教程

GymEye 希望提供一种更加直接、高效的方式：

用户只需举起手机扫描器械，即可立刻获得该器械的完整教学信息。

---

## 核心功能

### 1. 健身器械实时识别

* 使用手机摄像头扫描器械
* AI 模型自动识别器械类别
* 支持多种健身器械识别
* 在屏幕中实时显示识别结果和置信度

### 2. 器械详情展示

识别完成后，系统自动展示：

* 器械名称
* 主要训练部位
* 适合人群
* 使用步骤
* 注意事项
* 常见错误动作

### 3. 教学视频播放

* 每个器械配套标准动作视频
* 支持在线播放
* 视频时长控制在 30 秒到 1 分钟内
* 让用户快速理解如何正确使用器械

### 4. 异常状态提示

* 未识别到器械时提示重新扫描
* 网络异常时提示稍后重试
* 视频加载失败时提供备用说明文字

### 5. 后续可扩展功能

* 动作姿态识别
* 动作纠错
* 个性化训练计划推荐
* 收藏常用器械
* 历史记录
* 微信小程序版本

## 项目架构

### 前端

* Android 原生 App
* CameraX：摄像头调用
* ExoPlayer：视频播放
* Retrofit / OkHttp：网络请求
* Material Design：界面设计

### AI 模型

* YOLOv8：器械目标检测
* Roboflow：数据标注与管理
* TensorFlow Lite：移动端部署
* 模型量化：减少模型体积和手机发热

### 后端

* RESTful API
* 数据库存储器械信息和视频链接
* 云存储保存视频资源
* 缓存机制减少重复请求

## 用户使用流程

1. 打开 GymEye
2. 打开摄像头扫描器械
3. 系统识别器械类型
4. 获取器械详细信息
5. 查看训练部位和使用说明
6. 播放教学视频
7. 开始训练

## 项目优势

### 垂直领域识别更精准

与通用识图 App 不同，GymEye 只专注于识别健身器械，因此能够在健身场景下获得更高的识别准确率。

### 提供标准化教学内容

相比短视频平台内容杂乱、娱乐化严重，GymEye 提供的是结构化、标准化的教学信息，帮助用户快速找到真正有用的内容。

### 更符合健身场景

GymEye 是一个“用完即走”的工具，用户扫完器械、学会动作后就可以立即开始训练，不会被无关内容打断。
